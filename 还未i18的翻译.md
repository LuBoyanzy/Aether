# 待补充的多语言翻译

以下词条在简体中文 (zh-CN) 中已翻译，但在其他语言中尚未翻译（目前回退显示为英文）。需要后续在对应的 `.po` 文件中进行补充。

## 导航栏 & 通用 (Navbar & Common)

- [ ] `msgid "Containers"`
  - **zh-CN**: "容器"
  - **Context**: 导航栏链接

- [ ] `msgid "User Actions"`
  - **zh-CN**: "用户操作"
  - **Context**: 导航栏菜单 / 命令面板

- [ ] `msgid "Language"`
  - **zh-CN**: "语言"
  - **Context**: 导航栏语言选择 / 设置页面

- [ ] `msgid "Toggle theme"`
  - **zh-CN**: "切换主题"
  - **Context**: 主题切换按钮

- [ ] `msgid "All Systems"`
  - **zh-CN**: "所有客户端"
  - **Context**: 导航栏 Grid 图标 (首页链接)
  - **Note**: 该词条可能在部分语言中已存在，需检查确认。

## 操作指南

1. 运行 `npm run sync` 提取最新词条。
2. 打开对应语言的 `.po` 文件 (例如 `src/locales/es/es.po`)。
3. 搜索上述 `msgid`。
4. 在 `msgstr` 中填入对应的翻译。
5. 再次运行 `npm run sync` 编译生效。
